☕ Tea vs Coffee Analytics Dashboard

[Power BI] [DAX] [Data] [Status]

------------------------------------------------------------------------

📌 Project Overview

The Tea vs Coffee Analytics Dashboard is an interactive Power BI report
designed to analyze consumer preferences, consumption patterns, and
lifestyle factors influencing beverage choices.

------------------------------------------------------------------------

🎯 Objective

-   Compare tea vs coffee consumption patterns
-   Analyze customer satisfaction and taste preferences
-   Identify demographic and regional trends
-   Understand lifestyle impact (sleep, stress, work type)

------------------------------------------------------------------------

🛠️ Tech Stack

-   Power BI Desktop
-   Power Query
-   DAX
-   Data Modeling

------------------------------------------------------------------------

📂 Dataset Description

Includes: - Respondents (15K+)
- Beverage preference
- Cups per day
- Monthly spending
- Satisfaction & taste scores
- Demographics & lifestyle factors

------------------------------------------------------------------------

📊 Dashboard Pages

Overview | Analysis | Insights

------------------------------------------------------------------------

📈 Business Impact

-   Improve product strategy
-   Understand customer behavior
-   Optimize pricing decisions
-   Enable data-driven insights
